• 布局
  • align-content
content-center

使用 content-center 应用align-content属性设置 Flex 容器居中对齐，所有子元素在容器中轴线上居中排列。

content-start

使用 content-start 应用align-content属性设置 Flex 容器起始对齐，所有子元素从容器起始边紧凑排列。

content-end

使用 content-end 应用align-content属性设置 Flex 容器末尾对齐，所有子元素向容器末尾边紧凑排列。

content-between

使用 content-between 应用align-content属性设置 Flex 容器两端对齐，首尾元素贴边，剩余空间平均分配到元素之间。

content-around

使用 content-around 应用align-content属性设置 Flex 容器环绕对齐，每个元素两侧拥有相等间距，元素间距是侧边距的两倍。

content-evenly

使用 content-evenly 应用align-content属性设置 Flex 容器均匀对齐，所有间距（包括元素与容器边界之间）都完全相等。
