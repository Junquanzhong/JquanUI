  • align-items
items-center

使用 items-center 应用align-items属性设置 Flex 容器交叉轴居中对齐，所有元素在交叉轴方向上居中对齐。

items-start

使用 items-start 应用align-items属性设置 Flex 容器交叉轴起始对齐，所有元素的顶部/左侧对齐到交叉轴的起始线。

items-end

使用 items-end 应用align-items属性设置 Flex 容器交叉轴末尾对齐，所有元素的底部/右侧对齐到交叉轴的末尾线。

items-baseline

使用 items-baseline 应用align-items属性设置 Flex 容器进行基线对齐，所有元素以其内部文本的基线为准进行对齐。

items-stretch

使用 items-stretch 应用align-items属性设置 Flex 容器进行拉伸填满，所有元素（在未设高度/宽度时）被拉伸以填满容器的整个交叉轴空间。
