包含

使用 object-contain 来指定元素如何适应容器的高度与宽度

保持原比例，完整缩放进容器，不一定填满

不会出现裁剪，不失真,可能左右或上下留空白。

覆盖

使用 object-cover 来指定元素如何适应容器的高度与宽度

保持比例，填满整个容器，不一定会完整显示

填充满，不失真，适合封面图，多余部分会被裁剪掉。

充满

使用 object-fill 来指定元素如何适应容器的高度与宽度

强制拉伸填满容器，不保持比例

填满无留白，可能变形。

无充满方式

使用 object-none 来指定元素如何适应容器的高度与宽度

不缩放，按原图大小放入容器

只显示能容下的部分，不会缩放也不会适应,常用于“裁剪预览”类场景。

缩小

使用 object-scale-down 来指定元素如何适应容器的高度与宽度

有缩放行为，但只会缩小图像，不会放大

更像 contain 的“保守版本”：比容器大就缩小，比容器小就保持原样。

注意缩小的概念，是图像自适应放大时，尺寸超过容器时，才会缩小，如果有给图像固定尺寸，这个尺寸大于容器，则可能会溢出容器。
