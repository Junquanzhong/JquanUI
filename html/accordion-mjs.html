参数文档
参数类别 参数名称 / HTML 属性 类型 / 可选值 默认值 说明
JavaScript 选项 mode string 'multiple' 设置手风琴的模式。'single' 表示同时只能展开一个面板；'multiple' 表示可以同时展开多个面板。
defaultOpen boolean false 是否在初始化时默认展开第一个面板（仅在 mode 为 'single' 时有效）。
animationDuration number 300 展开和收起动画的持续时间，单位为毫秒（ms）。
options object {} 在通过 new Accordion(element, options) 实例化时，用于传入上述所有配置的对象。
HTML 数据属性 data-accordion string null 在 HTML 标签上设置，用于覆盖 JavaScript 中的 mode 选项。值应为 'single' 或 'multiple'。
data-accordion-open boolean false 在 HTML 标签上设置（存在即表示 true），用于覆盖 defaultOpen 选项。
data-accordion-duration number 300 在 HTML 标签上设置，用于覆盖 animationDuration 选项，值应为毫秒数。
data-accordion-initialized string 'true' 由组件自动添加的属性，用于标记手风琴元素已被初始化，防止重复初始化。开发者无需手动设置。
CSS 类 / 选择器 .is-open class - 动态添加到 .accordion-item 上的类，表示该项当前处于展开状态。用于控制图标的旋转等。
.is-fully-opened class - 动画结束后动态添加到 .accordion-item 上的类，用于将内容的 max-height 设为 none，以适应动态内容变化。
.focus-visible class - 当通过键盘（Tab键）使 .accordion-header 获得焦点时，动态添加的类，用于显示自定义的焦点样式。
CSS 变量 --accordion-duration 300ms 由 JavaScript 根据 animationDuration 选项动态设置的 CSS 变量，用于控制 transition-duration，确保 CSS 和 JS 动画时长同步。
DOM 结构 .accordion-item selector - 每个手风琴项的容器。必须包含 .accordion-header 和 .accordion-content。
.accordion-header selector - 手风琴的触发器，通常是一个 <button> 元素。
.accordion-content selector - 内容面板的包裹层，用于 max-height 动画。
.accordion-content-inner selector - 内容面板的内层包裹，用于存放实际内容和 padding，方便 JS 测量其真实高度（scrollHeight）。
