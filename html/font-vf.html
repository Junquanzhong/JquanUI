衬线：font-srif-{ number }(1|0)

真斜体：font-ital-{ number }(1|0)

光学调整：font-opsz-{ size }(xs|sm|md|lg|xl|2xl|3xl|4xl|5xl|6xl|7xl|8xl) 尺寸同字号，opsz 是可变字体中最“智能”的特性之一，它允许字体根据文字的显示尺寸，自动调整其字重、字宽、衬线粗细、字间距等细节，以达到在不同尺寸下最佳的阅读体验。
