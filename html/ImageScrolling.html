参数
参数名 类型 默认值 说明 用法示例
container string | HTMLElement 无（必填） 滚动组件的容器元素或CSS选择器。容器内必须包含一个ul或带有class="scroll-content"的元素作为滚动内容。 container: '#infiniteScrollContainer'
或
container: document.querySelector('#infiniteScrollContainer')
autoScroll boolean true 是否启用自动滚动功能。如果设置为false，则只能通过拖动滚动。 autoScroll: true
speed number 1 自动滚动的速度，范围为1-10。值越大，滚动越快。 speed: 3
visibleItems number 3.5 可视区域显示的项目数量，可以是小数，如3.5表示显示3个完整项目和一个半项目。 visibleItems: 3.5
direction string 'horizontal' 滚动方向，可选值：'horizontal'（水平）或'vertical'（垂直）。 direction: 'horizontal'
gap number 16 项目之间的间距，单位为像素(px)。 gap: 20
pauseOnHover boolean true 鼠标悬停或触摸时是否暂停自动滚动。 pauseOnHover: true
公共方法
方法名 参数 返回值 说明 用法示例
toggleAutoScroll enable: boolean void 启用或禁用自动滚动功能 infiniteScroll.toggleAutoScroll(true);
setSpeed speed: number void 设置自动滚动的速度（1-10） infiniteScroll.setSpeed(5);
setVisibleItems count: number void 设置可视区域显示的项目数量 infiniteScroll.setVisibleItems(2.5);
pauseAutoScroll 无 void 暂停自动滚动 infiniteScroll.pauseAutoScroll();
resumeAutoScroll 无 void 恢复自动滚动 infiniteScroll.resumeAutoScroll();
destroy 无 void 销毁组件，移除所有事件监听器 infiniteScroll.destroy();
