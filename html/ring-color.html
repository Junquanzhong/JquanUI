语义化

使用 ring-* 类来添加轮廓。

  • ring-secondary

  • ring-success

  • ring-warning

  • ring-danger

  • ring-important

  • ring-special

  • ring-black

  • ring-white

:hover

使用 .hover-ring-*-* 在环颜色类前加一个.hover-来设置悬浮时的环颜色

  • hover-ring-blue-500

:focus

使用 .focus-ring-*-* 在环颜色类前加一个.focus-来设置聚焦时的环颜色

  • focus-ring-blue-500

:focus-visible

使用 .focus-visible-*-* 在环颜色类前加一个.focus-visible来设置聚焦时的环颜色

  • focus-visible-ring-blue-500

偏移环颜色

使用 .ring-offset-{color}-{50-900} 在环颜色中加一个offset来设置偏移环颜色，其他用法一致

  • ring-offset-red-500
