悬浮提示器

通过添加属性：data-tips 来控制提示内容

通过添加属性：data-tips-pos 来控制提示位置(top、bottom、left、right)

通过添加属性：tips-*(色彩) 来控制提示器颜色，比如tips-blue，色彩名同色彩系统

参数

参数 类型 默认值 说明
type string info 提示类型，可选值有：info、success、error、warning、loading
timeout number 3000 延时关闭时间，单位毫秒
autoClose boolean true 是否自动关闭，默认true,注意在type为loading的时候自动关闭为false
showClose boolean false 是否显示关闭图标，默认为false不显示
content string 这是一条提示消息 提示的内容
onClose function null 关闭的回调函数
