选项卡 MJS 组件

基于 ES6 模块的响应式选项卡组件

组件演示

选项卡组件功能

这是一个基于 ES6 模块的响应式选项卡组件：

  • 支持两种切换模式：点击切换和浮动切换
  • 完全响应式设计，适配各种屏幕尺寸
  • 平滑的切换动画效果
  • 可自定义默认激活项
  • 移动端友好的交互体验
  • 基于 ES6 模块，支持按需导入

使用说明

要使用此选项卡组件，您需要：

  1. 引入组件的 CSS 文件
  2. 按需导入 ES6 模块
  3. 初始化组件并传递配置参数
// 按需导入选项卡组件 import Tabs from './modules/tabs.mjs'; // 初始化选项卡 const tabs = new Tabs({ container: '#tabsContainer', switchMode: 'click', defaultActive: 0, animation: true, themeColor: '#3498db' });

API 文档

组件提供以下 API 方法：

  • activateTab(index) - 激活指定索引的选项卡
  • addTab(title, content) - 动态添加选项卡
  • removeTab(index) - 移除指定索引的选项卡
  • setThemeColor(color) - 设置主题颜色
  • updateConfig(config) - 更新组件配置
  • destroy() - 销毁组件实例

响应式设计

此选项卡组件采用了完全响应式设计：

  • 在桌面端，选项卡水平排列
  • 在平板设备上，布局自动调整
  • 在手机上，选项卡可以水平滚动
  • 触摸设备上优化了交互体验

多实例演示

多实例支持

组件支持在同一个页面创建多个独立的选项卡实例。

每个实例都有自己的配置和状态，互不干扰。

独立配置

可以为每个实例设置不同的：

  • 切换模式
  • 默认激活项
  • 主题颜色
  • 动画效果
