为什么选择 JquanUI?

不仅仅是一个 CSS 文件，而是一套完整的设计系统。

中式传统配色

内置 金盏花黄、翡翠绿、天霁蓝 等数十种富有诗意的中国传统色彩变量。

完美暗黑模式

基于 CSS 变量的自动暗黑模式。只需给 body 添加 theme-dark 类，所有组件自动适配。

3D 与 动效

内置 3D 翻转卡片、平滑过渡、加载动画等高级 CSS 特效，无需编写额外 JavaScript。

精美组件库

开箱即用的 UI 元素，支持完全自定义。
