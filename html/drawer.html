JquanUIex Drawer

生产级、高性能、无障碍优先的侧边抽屉模块。

ES Module A11y Focus Trap 60FPS CSS Animation
基础方向

支持 Left, Right, Top, Bottom 四个方向，只需修改 data-position。

交互与无障碍 (A11y)

打开此抽屉后，尝试使用 Tab 键。你会发现焦点被锁定在抽屉内部，不会跳回主页面，这是生产级组件的标准。

配置参数 (Data Attributes)
属性名 值 / 类型 说明
data-drawer-target ID String 触发器属性，指向抽屉元素的 ID。
data-position right | left | top | bottom 抽屉滑出的方向。默认为 right。
data-drawer-close Boolean 放在抽屉内部的按钮上，点击自动关闭当前抽屉。
data-backdrop true | false 是否显示遮罩层。默认为 true。
设置

这是标准的右侧抽屉，常用于设置面板或详情预览。
