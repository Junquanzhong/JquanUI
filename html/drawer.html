使用参数

抽屉组件 API 参数参考

所有参数均通过设置在触发按钮 <button> 上的 data-* 属性来配置。抽屉容器本身（<div>）无需设置任何特殊属性，只需一个与 data-drawer-target 匹配的 id。

参数 可选值 默认值 描述
data-drawer-target string (ID) 必填 指定该按钮要控制的抽屉容器的 id。这是建立连接的唯一必需参数。
data-drawer-position "left", "right" "right" 定义抽屉从屏幕的哪一侧滑入。
data-drawer-disable-mask (无值) 未设置 布尔属性。只要存在此属性，遮罩层就会被禁用。用户无法通过点击背景来关闭抽屉，只能通过关闭按钮或按 ESC 键。
data-drawer-body-lock "true", "false" "true" 控制在抽屉打开时是否锁定主页面的滚动。设置为 "false" 时，用户仍可与背景页面交互并滚动。
data-drawer-close (无值) N/A 布尔属性。此属性用于放在抽屉内部的关闭按钮上。点击带有此属性的元素会关闭当前打开的抽屉。

关闭抽屉的方式

无论配置如何，用户始终可以通过以下方式关闭抽屉：

  • 点击抽屉内部任何一个带有 data-drawer-close 属性的元素。
  • 在抽屉获得焦点时按下 ESC 键。
  • 点击背景遮罩层（除非通过 data-drawer-disable-mask 禁用了此功能）。
