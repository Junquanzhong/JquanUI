组件参数
参数名 类型 默认值 说明
container String/DOM元素 '.carousel' 轮播图容器的选择器或DOM元素
direction String 'horizontal' 轮播方向，可选值：'horizontal'(水平)或'vertical'(垂直)
rtl Boolean false 是否从右到左轮播(仅在horizontal模式下有效)
autoplay Boolean true 是否自动播放轮播图
interval Number 3000 自动播放的时间间隔(毫秒)
speed Number 500 切换动画的过渡速度(毫秒)
indicators Boolean true 是否显示指示器(圆点)
arrows Boolean true 是否显示左右箭头导航
pauseOnHover Boolean true 鼠标悬停时是否暂停自动播放
infinite Boolean true 是否启用无限循环模式
onChange Function null 幻灯片切换时的回调函数，参数为(索引, 幻灯片元素)
