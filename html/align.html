核心场景：与 text-align: justify 配合使用

当文本设置为两端对齐 ( justify ) 时，默认情况下最后一行通常是左对齐的。text-align-last 类允许我们改变这一点。

默认:text-align-last: auto

孤独是你长久的朋友，爱只会偶尔来敲门。我们应当与孤独同生，弹琴、看书、买菜，对日子诚恳。一生有太多的遗憾与失去，我们要好好打扫悔恨，种新的花，忘记旧的人，掌好夜深的每一盏灯，爱来时窃窃听闻。

🛑 结果：auto 值使得最后一行遵循默认的对齐方式（对于 justify 来说，通常是 `left`），看起来不对称。

居中对齐最后一行:text-align-last-center

孤独是你长久的朋友，爱只会偶尔来敲门。我们应当与孤独同生，弹琴、看书、买菜，对日子诚恳。一生有太多的遗憾与失去，我们要好好打扫悔恨，种新的花，忘记旧的人，掌好夜深的每一盏灯，爱来时窃窃听闻。

✅ 结果：center 值让最后一行居中，实现了更均衡、更美观的排版效果，常用于诗词或引用。

方向感知性：start vs end

start 和 end 是逻辑属性，它们的对齐方向取决于文本的书写方向（direction），这对于国际化（i18n）非常重要。

从左到右 (LTR, dir="ltr")

In LTR text, start means left.

In LTR text, end means right.

从右到左 (RTL, dir="rtl")

In RTL text, start means right.

In RTL text, end means left.
